# cloudfoundry_org (Resource)

Creates a Cloud Foundry Organization

## Example Usage

```terraform
resource "cloudfoundry_org" "org" {
  name      = "tf-test"
  suspended = false
  labels = {
    env = "test"
  }
  annotations = {
    env-ann = "test-ann"
  }
}
```
